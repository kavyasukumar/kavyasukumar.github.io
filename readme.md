# Kavya Sukumar Portfolio

This project is a high-performance, typography-driven portfolio built with React, TypeScript, and Vite.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. Run Development Server:

```bash
npm run dev
```

3. Build for Production:

```bash
npm run build
```

## Project Structure
- `src/App.tsx`: Main portfolio component and logic.
- `src/main.tsx`: React entry point.
- `index.html`: Base HTML template.
- `vite.config.ts`: Build and development configuration.
